# Изображения карт для Blackjack

Эта папка должна содержать изображения карт для игры в Blackjack.

## Формат файлов

Изображения должны быть названы в формате: `{ранг}{масть}.png` (или `.jpg`/`.jpeg`)

## Необходимые файлы

### Масти:
- ♠ (пики)
- ♥ (черви)
- ♦ (бубны)
- ♣ (трефы)

### Ранги:
- A (туз)
- K (король)
- Q (дама)
- J (валет)
- 10, 9, 8, 7, 6, 5, 4, 3, 2

## Примеры названий файлов:

- `A♠.png` - туз пик
- `K♥.png` - король червей
- `Q♦.png` - дама бубен
- `J♣.png` - валет треф
- `10♠.png` - десятка пик
- `9♥.png` - девятка червей
- и т.д.

## Всего нужно 52 файла (13 рангов × 4 масти)

## Формат изображений

- Рекомендуемый формат: PNG (с прозрачностью) или JPG
- Рекомендуемый размер: 360×520 пикселей (или пропорционально)
- Если изображение не найдено, бот будет использовать программную генерацию как fallback

## Где найти изображения карт

Вы можете:
1. Скачать готовые изображения карт из интернета
2. Создать свои изображения
3. Использовать любые изображения карт, соответствующие формату названий

## Примечание

Если изображение карты не найдено, бот автоматически сгенерирует простое изображение программно.
